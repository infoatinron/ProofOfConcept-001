ProofOfConcept-001 is een CSharp solution die met AI wordt ontwikkeld, dit om in een proof of concept aan te tonen dat gebruik van AI een krachtige co-pilot is die de ontwikkelingssnelheid 10x verhoogt, mits geleid door een sterke software architectuur en menselijke verificatie.

ProofOfConcept-001 is een WebAPI solution met clean architecture, de juiste patterns, interfaces, DTO's en de UnitOfWork. Via AI worden het use case diagram en de individuele de use cases in XML omgezet en gevoed aan AI. Dit genereert alle classes, interfaces, de logica, de entities en de controller in de API.
